## Project Description
This project is an Oracle Database Management System developed for a Real Estate Business. It manages property owners, agents, customers, properties, visits, sales, rentals, and payments using SQL and PL/SQL.

## Technologies Used

- Oracle SQL Developer
- SQL
- PL/SQL

## How to Run
1. Create the Oracle user.
2. Run `create_tables.sql`.
3. Run `create_sequences.sql`.
4. Run `insert_data.sql`.
5. Run `queries.sql`.
6. Run `views_indexes.sql`.
7. Run `plsql_programs.sql`.
8. Run `triggers.sql`.
